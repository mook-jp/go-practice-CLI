# Go-Practice-CLI

## Description
このパッケージはGoの学習を進める時の不便さを解消するために作成しています。

学習をするときに不便に感じていること:
- mainから始めるためにはフォルダを分離する必要がある。
  学習するときは本やサイトをもとに学習するため大体順番にフォルダを作成するため、
  先頭に数字をつけた頭文字をつけるような構成で学習を進める。
  毎回をそのルールに基づいてフォルダを作成して、メインファイルを作成するのがめんどくさい。

## Installing
gopractice-cli はgoを学習するためのツールを、
あなたのパソコンにインストールします。

```
go install github.com/mook-jp/gopractice-cli@latest
```

## Usage
### 新しいプロジェクトを開始する
プロジェクトを開始するとディレクトリ内に、`.gop`ディレクトリが生成されます。

```bash:プロジェクト開始
gopractice-cli init [path]
```

### 新しいプラクティスディレクトリを生成する
練習用のディレクトリを生成する。
オプションを付与しない場合は、プロジェクトのルートディレクトリに新しいディレクトリが生成されます。

```bash:新しいディレクトリを生成
gopractice-cli new [name]
```

名前を指定しない場合は、 `00_Lesson` という名前で作成する。

option:
- `-i(--intaeractive)` : 対話モードで実行
- `sec-no`: 作成するセクション番号を指定
- `--subsec <name>`: ディレクトリを作成する。サブセクションを名前指定
- `--subsec-no <number>`: ディレクトリを作成する。サブセクションを番号指定
- `-t(--template) <name>`: テンプレートファイルを指定する
- `--no-template`: テンプレートファイルを作成しない
- `--no-readme`: README.mdを作成する・しない(デフォルト=true)

### 各セクション番号を整理する
各セクション番号を01~99（001~999,0001~9999）でディレクトリ名を変更する。
ソートされるディレクトリは `数字_名称` の構成で作成されているディレクトリを対象としてソートします。
`dir path` および オプション指定がない場合は、直下の各セクションディレクトリを整理する。
※ `dir path` をフルパスで指定している場合は、プロジェクトフォルダ以外でも利用可能

```bash:セクション番号を整理
gopractice-cli sort [dir-path]
```

sample:
変更を行った場合の例を下記する。
|変更前|変更後|
|----|----|
|01_aaa|01_aaa|
|02_bbb|02_bbb|
|04_ddd|03_ddd|
|09_iii|04_iii|

option: 
- `subsec <name>`: 指定されたサブセクション内をソートします。
- `subsec-no <number>`: 指定されたサブセクション番号内をソートします。

### テンプレートファイルを作成する
利用するテンプレートファイルを作成する。
オプションをしていない場合は、グローバルに作成する。
※ グローバルディレクトリ は `~/.gop` に作成されます。

```bash:テンプレートディレクトリを作成する
gopractice-cli template <name> [テンプレート文字列]
```

option:
- `-f(--file) <path>`: ファイル名を指定してテンプレートして登録する
